# Mass Software Installer for Windows

Утилита для массовой автоматической установки популярного ПО в системах Windows.

## Описание

Этот PowerShell скрипт позволяет одним действием установить несколько предварительно заданных программ с открытым исходным кодом. Использует пакетный менеджер Chocolatey.

## Требования

- Windows 7+
- PowerShell 5.0+
- Права администратора

## Поддерживаемые программы

- Notepad++ (текстовый редактор)
- 7-Zip (архиватор)
- VLC (медиаплеер)
- Git (система контроля версий)
- SumatraPDF (PDF-просмотрщик)

## Как использовать

### Способ 1: Простой запуск

1. Скачайте все файлы из репозитория в одну папку
2. Запустите `Enable-Script-Execution.bat` (требуется один раз)
3. Запустите PowerShell **от имени администратора**
4. Перейдите в папку со скриптом: `cd C:\path\to\folder`
5. Выполните: `.\Mass-Installer.ps1`

### Способ 2: Вручную через PowerShell

1. Откройте PowerShell **от имени администратора**
